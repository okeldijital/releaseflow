# PX-200.1 — Application Shell Implementation

**Sprint:** PX-200 Phase 1
**Date:** 2026-06-28
**Status:** Complete

---

## What Changed

### Sidebar (`packages/ui/src/navigation/sidebar.tsx`)
- Smooth slide transition (`transition-transform duration-300`) for mobile open/close
- Escape key dismisses sidebar on mobile (`window.innerWidth < 1024`)
- Auto-close on navigation click (mobile only)
- Active state uses `rounded-lg` + `shadow-sm` for elevation
- Backdrop uses `bg-black/20 backdrop-blur-sm` with fade transition
- Icon size bumped to `h-5 w-5` for better touch targets

### Topbar (`packages/ui/src/navigation/topbar.tsx`)
- Integrated Search (`⌘K` helper badge) — controlled via `searchValue` state
- Notification bell with unread count badge (`bg-danger-500` ring style)
- Command palette trigger button with `⌘K` keyboard hint
- Global keyboard shortcut: `⌘K`/`Ctrl+K` for command palette
- Global keyboard shortcut: `⌘\`/`Ctrl+\` for sidebar toggle

### AppShell (`packages/ui/src/layouts/app-shell.tsx`)
- Skip-to-content link (`sr-only focus:not-sr-only`) for keyboard accessibility
- Keyboard shortcut registration (`⌘K` for palette, `⌘\` for sidebar)
- Explicit `tabIndex={-1}` on main content for skip link target
- Passes `onSearch`, `notificationCount`, `onOpenNotifications`, `onOpenCommandPalette` to Topbar

### New: `Page` layout component
```tsx
<Page maxWidth="max-w-5xl">
  {children}
</Page>
```
Standardized page container with configurable max-width.
