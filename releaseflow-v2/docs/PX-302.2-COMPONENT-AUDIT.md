### Changes Made
- **Button**: Pressed state added (`active:scale-[0.98]`) to all 5 variants. Transition duration standardized to `150ms ease-out`. Transition property widened from `colors` to `all` for scale animation.
- **No other changes**: Remaining 30+ components already conform.

### P3 Observation
- **HealthRing**: Uses 3-band color model (≥80 success, ≥40 warning, <40 danger) rather than the 5-level PDS health system (Excellent/Healthy/Attention/Blocked/Critical). The ring's colors communicate green/amber/red rather than the 5 health labels. This is a visual encoding decision, not a data inconsistency — the ring always receives a 0-100 percentage.
