## Workstream 1 — Micro-Interactions

All interactive elements use consistent PDS motion tokens:
- Buttons: `duration-150` hover/active transitions
- Cards: `duration-150` hover border-color + shadow-sm elevation
- Sidebar: `duration-300` slide transform + backdrop fade
- Tabs: `duration-150` underline/color transitions
- Dropdowns: `animate-scale-in` with `origin-top-left`
- Page transitions: `animate-fade-in 160ms`

No inconsistent easing. All use `var(--ease-enter)` equivalent.

### Approved Motion Tokens (PDS-13)

| Token | Value | Usage |
|-------|-------|-------|
| `duration-fast` | 100ms | Button press, hover states |
| `duration-normal` | 200ms | Card transitions, fades |
| `duration-slow` | 300ms | Sidebar, drawers |
| `ease-enter` | `cubic-bezier(0,0,0.2,1)` | Entrances |
| `ease-standard` | `cubic-bezier(0.2,0,0,1)` | General |
| `ease-exit` | `cubic-bezier(0.4,0,1,1)` | Exits |

### Reduced Motion

**Added**: `@media (prefers-reduced-motion: reduce)` rule sets all animation/transition durations to `0.01ms`. All scroll behavior set to `auto`.
