# PX-306.2 — Information Architecture

**Date:** 2026-06-28

---

## Reading Sequence

The Operations Center now follows an information architecture designed for decision-making, not data exploration.

```
SITUATION (Hero)
    ↓
ASSESSMENT (Two-column grid)
    ↓
ACTION (Immediate recommendations)
    ↓
EVIDENCE (Metrics inline + Active Releases table)
    ↓
HISTORY (Recent Activity)
```

Each level answers one operational question.

---

## Sections

| # | Section | Question Answered | Source |
|---|---------|-------------------|--------|
| 1 | **Hero** | What is the operational state? | `generateBriefing()` from real data |
| 2 | **Assessment** | What is the current condition? | `aggregateHealthPct`, `currentStage`, `daysUntilRelease` |
| 3 | **Immediate Actions** | What should I do right now? | `generateActions()` from alerts/blocked/deadlines |
| 4 | **Metrics** | What are the numbers? | `pulseMetrics.activeReleases`, `blockedReleases`, `shippedThisMonth` |
| 5 | **Active Releases** | Which releases need attention? | `releases` from `useOperationsCenter` |
| 6 | **Attention** (conditional) | What is mission-critical? | `alerts`, `blockedItems`, `deadlines` |
| 7 | **Activity** | What happened recently? | `activities` from `useOperationsCenter` |

---

## Layers

### Hero
- Date as H1 (40px/medium/tight leading)
- Briefing (3 lines max, plain language)
- No "Operations Center" page title

### Assessment
- Two-column editorial grid
- Health / Confidence / Current Stage / Deadline
- Max width 640px
- Typography establishes hierarchy

### Immediate Actions
- Max 3 actions
- Priority-ordered: alerts → blocked → deadlines
- One sentence per action
- NOW timestamp right-aligned
- Max width 640px

### Metrics
- Inline format (not cards)
- 16px below actions (no separator)
- 3 numbers: active, blocked, shipped
- Max width 640px

### Active Releases
- Full-width table (extends beyond reading column)
- Decision support tool
- Click any row to open release

### Attention
- Conditional — only when items exist
- Alerts, blocked, deadlines
- Critical signal — uses background highlights

### Activity
- Bottom of page
- Supporting evidence
- Lower contrast, smaller text

---

## Vertical Rhythm

```
Hero (24px) Assessment (16px) Actions (20px) Metrics (40px) Table (24px) Activity
```

---

## What Does NOT Appear

- "Operations Center" repeated page title
- Badges and pills
- Decorative containers
- Section underlines or dividers
- Glass or gradient effects
- Card backgrounds (except in attention panel for critical signal)
