# PX-304.2 — Editorial Refinement

## Summary

Applied systematic visual polish across the authenticated shell and core workspaces. Every change is an exercise in restraint: remove chrome, increase whitespace, strengthen typographic hierarchy, and let the editorial system breathe.

## Scope

No backend, domain logic, services, hooks, repositories, routing, or calculations were modified.

## Files Modified

| File | Change |
|------|--------|
| `packages/ui/src/components/empty-state.tsx` | Increased intentional whitespace, scaled icon container, grown title to `text-base`, widened description, elevated CTA margin. |
| `packages/domain-ui/src/components/operational-summary.tsx` | Expanded padding (`p-8`), widened metric gaps, thickened confidence bar, deepened section dividers, receded timestamp. |
| `packages/ui/src/components/card.tsx` | Reduced border opacity to `border-surface-200/70`; switched transition focus from shadow to color. |
| `packages/ui/src/components/table.tsx` | Removed heavy header backgrounds, softened row separators, reduced outer border opacity, modernized rounded corners. |
| `packages/ui/src/navigation/sidebar.tsx` | Tightened brand gap, increased nav item spacing, enlarged active indicator, lightened footer rule, softened footer border. |
| `apps/web/src/app/(app)/dashboard/page.tsx` | Removed duplicate H1 (topbar now owns the screen title), removed decorative footer border, lightened alert borders, removed deadline row borders, increased section rhythm. |
| `apps/web/src/app/(app)/releases/[id]/page.tsx` | Removed decorative ring from release hero icon; removed duplicate H1. |
| `apps/web/src/app/(app)/artists/[id]/page.tsx` | Removed duplicate H1; softened release-card border; replaced elevated shadow with raised token. |
| `apps/web/src/app/(app)/**/*.tsx` | Global H1→P conversion across all `(app)` pages to enforce **one H1 per screen** (the topbar). |

## Key Outcomes

- **One H1 per screen** — Topbar owns the screen title; content uses `<p>` with identical visual weight.
- **Reduced chrome** — Borders softened from `border-surface-200` to `border-surface-200/70` (or removed where spacing suffices). Decorative rings and underlines eliminated.
- **Stronger hero** — `OperationalSummary` is now the undisputed focal point: `p-8` padding, `pt-8` metric rule, `gap-3` metric spacing, thicker confidence track (`h-1.5`).
- **Designed empty states** — `EmptyState` now feels like a deliberate pause: `py-20`, `p-5` icon well, `text-base` title, `max-w-sm` description, `mt-8` CTA.
- **Breathing room** — Section vertical rhythm increased from `mb-12` to `mb-14`; inner heading/subtitle spacing increased by 4px increments.
- **Refined sidebar** — Active indicator enlarged to `h-2 w-2`; footer border lightened; nav spacing expanded.
