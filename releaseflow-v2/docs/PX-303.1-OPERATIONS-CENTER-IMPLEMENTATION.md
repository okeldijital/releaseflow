# PX-303.1 — Operations Center Premium Composition

**Date:** 2026-06-28
**Status:** Complete

---

## What Changed

### Hero — Executive Briefing
- Date moves above the title as a small overline — contextual frame without competing
- Title remains editorial (1.75rem, tracking-tight)
- New executive summary line: "3 healthy · 2 at risk · 1 shipped this month" — answers core questions in one sentence
- Primary action (New Release) top-right, visually dominant

### Operational Summary — Visual Anchor
- Section gap increased from `mb-8` (32px) to `mb-12` (48px) — more negative space around the summary
- Now reads aggregate health from `fetchOrgIntelligence()` which computes the actual average

### Attention Panel — First Destination for the Eye
- Reordered: Critical (alerts) → Blocked → Deadlines
- Alert cards now show entity type context ("· Campaign", "· Stage")
- Alert cards have hover shadow and the "View →" link appears only on hover (cleaner idle state)
- Blocked items use a red badge count (danger-100/danger-600) to distinguish from deadlines
- Deadlines use tabular numbers for consistent column alignment
- Deadline rows stripped of redundant type badge and priority badge — just the task name and date

### Active Releases — Operational Workspace
- Header includes a subtitle: "5 releases · Click any row to open"
- Table already renders correct columns (Release, Health, Stage, Deadline, Owner)

### Org Pulse — Executive Snapshot
- **Replaced 5 individual KPI cards with single inline stat bar** — one row with dots between
- Active | Blocked | Shipped — only the 3 most important metrics
- Lighter background (`bg-white/60`) — recedes behind the Operational Summary
- Removed "overdue deadlines" and "over budget" which were hardcoded to 0

### Activity — Supporting Evidence
- Lower contrast: dots changed from `primary-500` to `surface-300` (fades into background)
- Typography reduced to `text-xs text-text-500` for message, `text-[10px] text-text-400` for time
- Removed the "View" link per item — activity is reference, not action
- Max items reduced to 8

### Quick Actions + Footer — Single Row
- Quick Actions merged into the footer as inline text links (not buttons)
- Refresh + "Updated X ago" on the right
- Single border-top divides the page from its closing

### Visual Noise Removal
- Removed 5 section overline headers (replaced with contextual headings where needed)
- Removed 5 KPI cards with hard borders
- Removed 10 "View" links from activity items
- Removed redundant badges from deadline rows
- Removed unused `useRef` import
