# RFDS-005 — Shortcuts

**Status:** Active
**Version:** 1.0

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘K | Command palette |
| ⌘\ | Toggle sidebar |
| ⌘1 | Home (Operations Center) |
| ⌘2 | Releases |
| ⌘3 | Artists |
| Esc | Close overlay / dismiss drawer |

---

## Page Shortcuts

| Page | Shortcut | Action |
|------|----------|--------|
| Release Workspace | ⌘E | Edit release |
| Release Workspace | ⌘S | Advance stage |
| Artist Workspace | ⌘E | Edit artist |

---

## Rules

- Never override OS shortcuts (⌘W, ⌘Q, ⌘Tab)
- Only register shortcuts visible in the current context
- ⌘K always works
- Esc always dismisses the current overlay
