# PX-306.2 — Rhythm System

**Date:** 2026-06-28

---

## Uniform Spacing (Before)

```
32 → 32 → 32 → 32
```

Equal gaps create visual monotony. Every section feels equally important.

## Alternating Rhythm (After)

```
Hero
    24px        ← tight
Assessment
    16px        ← tighter
Actions
    20px        ← moderate
Metrics
    40px        ← wide
Active Releases
    24px        ← tight
Attention
    24px        ← tight
Activity
```

Alternating tight → wide → tight creates editorial flow and establishes hierarchy through rhythm alone.

## Rules

- Hero margin: 24px (close coupling to next content)
- Assessment margin: 16px (very tight — assessment flows directly from hero)
- Actions margin: 20px (moderate — separate from assessment but close to metrics)
- Metrics margin: 40px (wide — signals transition to evidence)
- Tables: 24px before, 0px between rows (border separates)
- Activity: no explicit margin — flows as part of history

## Implementation

Standard Tailwind classes:

| Gap | Class |
|-----|--------|
| 16px | `mb-4` |
| 20px | `mb-5` |
| 24px | `mb-6` |
| 32px | `mb-8` |
| 40px | `mb-10` |
| 48px | `mb-12` |

No arbitrary values. No `mt-[17px]`. No `p-[13px]`.

## Why This Matters

Spacing is communication. When sections have different significance, they need different breathing room. Uniform gaps say "these are all equally important." Alternating gaps say "the hero is the answer, the metrics are the proof."
