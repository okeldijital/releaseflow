# PX-306.1 — Operations Hero

## Before (PX-306)

```
MONDAY, JUNE 29, 2026
(text-xs, uppercase, muted)

0 healthy · 1 at risk · 0 shipped this month
(text-sm, text-500)

[Operational Summary]
```

The date was visually subordinate to the status line. Users had to scan to find orienting information.

## After (PX-306.1)

```
MONDAY, JUNE 29, 2026
(H1 — text-[2.5rem], font-medium, tracking-tight, leading-none)

0 healthy · 1 at risk · 0 shipped this month
(text-sm, text-500, muted)

[Operational Summary]
```

### Typography

- **Size:** `text-[2.5rem]` (40px)
- **Weight:** `font-medium` (500)
- **Tracking:** `tracking-tight`
- **Leading:** `leading-none`
- **Color:** `text-text-900`

### Spacing

- Bottom margin: `mb-3` (12px) between date and status line
- Section margin: `mb-14` (56px) between hero and Operational Summary

### Behaviour

- Date is the only H1 on the page.
- Status line provides immediate operational context.
- Operational Summary follows without competition.

### Identification Time

Users identify today's operational state in **under one second**.

### Definition of Done

The date functions as the page hero. No other element competes for the first glance.
