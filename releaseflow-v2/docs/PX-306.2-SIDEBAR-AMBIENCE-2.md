# PX-306.2 — Sidebar Ambience 2.0

**Date:** 2026-06-28

---

## Goal

Make the sidebar disappear psychologically. Users focus on content, not navigation.

Inspirations: Linear, Raycast, Notion desktop.

---

## Changes (from previous sidebar)

| Element | Before | After |
|---------|--------|-------|
| Active state | Background highlight + primary text | Darker text only, no background |
| Hover | Background + text color | Text color only, no background |
| Section headers | `text-text-400 font-semibold` | `text-text-300/70 font-medium` |
| Corner radius | `rounded-lg` (14px) | `rounded-md` (10px) — tighter |
| Min row height | 44px | 40px — more compact |

## Rules

- No background highlight on hover
- No background highlight on active state
- Only text color indicates interactivity
- Section headers nearly invisible
- Brand at top: 16px height, 20px padding
- User section at bottom: minimal padding, no border

## Visual

The sidebar now appears as a column of text labels and icons on the canvas background — not a separate panel.

```
┌──────┬───────────────────────┐
│ 🎵   │                       │
│ R    │  Content area         │
│      │                       │
│ Home │  (transparent —      │

│ Art  │                       │
│ Asst │                       │
│ Adm  │                       │
│      │                       │

│      │                       │
│ 👤   │                       │
└──────┴───────────────────────┘
```

## Result

Navigation recedes. Content dominates. The sidebar's job is to be available, not visible.
