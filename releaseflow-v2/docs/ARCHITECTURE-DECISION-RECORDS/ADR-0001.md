
## Adoption of the ReleaseFlow Golden Architecture

**Status**

Accepted

**Date**

2026-06-28

**Decision Makers**

ReleaseFlow Architecture Board

---

# Context

Following the ST-001 Engineering Audit, ReleaseFlow was found to contain architectural divergence caused by multiple independent implementation streams.

Examples included:

- Firestore accessed from pages
    
- Business logic inside React components
    
- Duplicate workflows
    
- Duplicate service implementations
    
- Inconsistent state ownership
    
- Missing tenant isolation
    
- Direct Firebase usage outside repositories
    

Although individual features functioned, the overall architecture had drifted away from a consistent design.

Without intervention, technical debt would continue to compound.

---

# Decision

ReleaseFlow adopts the Architecture Golden Standard as the governing architectural authority.

The following layered architecture becomes mandatory:

```
Presentation

↓

Application

↓

Domain

↓

Repository

↓

Infrastructure

↓

Firebase
```

All future development shall conform to this structure.

---

# Consequences

## Positive

Single architecture

Predictable codebase

Simpler onboarding

Improved testing

Reduced duplication

Better scalability

Improved security

Clear ownership

---

## Negative

Initial refactoring effort

Migration of legacy code

Temporary slowdown in feature delivery

---

# Constraints

Only repositories may access Firebase.

Business logic belongs exclusively in domain services.

Pages coordinate only.

Shared UI owns presentation.

State ownership is singular.

Every architectural change requires an ADR.

---

# Migration Strategy

Phase 1

Freeze feature development.

Phase 2

Recover repository layer.

Phase 3

Recover services.

Phase 4

Recover state ownership.

Phase 5

Recover security.

Phase 6

Resume feature development.

---

# Compliance

Every pull request shall be evaluated against:

- ARCHITECTURE-GOLDEN-STANDARD.md
    
- ENGINEERING-STANDARDS.md
    
- Relevant ADRs
    

If conflicts arise, the ADR history supersedes implementation.
