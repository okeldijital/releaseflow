# PX-306.1 — Hierarchy Recovery

**Date:** 2026-06-28
**Status:** Complete

---

## Changes Made

### Operations Center Hero

**Before**: Two competing elements — "Operations Center" (H1) + date string below. Neither was dominant.

**After**: Date is the hero at 40px/medium weight/tight tracking:

```
MONDAY, JUNE 29, 2026          [+ New Release]

0 healthy · 1 at risk · 0 shipped this month

────────────────────────────
Operational Summary
```

- "Operations Center" page title **removed** — the date now functions as the page identity
- Date set at `text-[2.5rem] font-medium tracking-tight` (~40px)
- Status line immediately below at `text-sm text-text-500`
- Primary action (New Release) stays top-right
- No duplicated page title anywhere

### Sidebar — Ambient

**Before**: Sidebar had explicit background (`bg-surface-50`), border (`border-r surface-200`), and footer separator (`border-t`). It visually competed with content.

**After**:
- **Background removed** — sidebar sits directly on the application canvas
- **Border removed** — no vertical separator between sidebar and content
- **Footer separator removed** — no `border-t` in zone 3
- **Hover states softened** — `bg-surface-100/20` instead of `bg-surface-100/30`
- **Active states softened** — `bg-primary-50/40` instead of `bg-primary-50/60`
- **Row height reduced** — `min-h-[40px]` instead of `min-h-[44px]`
- **Vertical padding reduced** — `py-1.5` instead of `py-2`

### Editorial Weight

**Before**: Typography hierarchy was flat — date and title were at similar sizes, competing for attention.

**After**: 3 clear levels:
1. **Date (40px/medium)** — unmistakable page identity
2. **Status line (14px/500)** — operational context
3. **Operational Summary (body)** — detailed state

Typography creates hierarchy. No borders, no containers, no sections.

### Hero-Before-Metrics

The correct reading order is restored:

```
1. Date (hero)
2. Status line
3. Operational Summary
4. Attention (if any)
5. Active Releases (table)
6. Org Pulse (inline stat)
7. Activity (supporting evidence)
```

Metrics are evidence, not the headline.

### Silence Preserved

No reintroduced:
- card borders
- section containers
- boxed panels
- heavy dividers
- gradients
- glass effects
- elevation on non-interactive elements
