# PX-304.1 — Before / After

## Context

The previous shell was a floating dashboard-style layout without a full-height fixed frame, inconsistent drawer behaviour, missing gesture support on mobile, and no page title in the top bar. Users described it as a "generic admin template."

## After — Application Shell

### Expanded desktop shell
![Expanded desktop shell](docs/PX-304.1-screenshot-expanded-desktop.png)

The sidebar occupies the full viewport height. Brand, navigation, and user/settings/sign-out zones are clearly delineated. The 280px expanded rail provides comfortable label+icon navigation with active-state indicators.

### Collapsed desktop shell
![Collapsed desktop shell](docs/PX-304.1-screenshot-collapsed-desktop.png)

72px icon rail with tooltips on hover. Content area fills the remaining width without overlap. Collapse preference persists via `localStorage`.

### Tablet layout
![Tablet layout](docs/PX-304.1-screenshot-tablet.png)

Overlay drawer with backdrop blur. The shell remains fixed while the drawer slides over content. Breadcrumbs and search stay accessible in the top bar.

### Mobile layout
![Mobile layout](docs/PX-304.1-screenshot-mobile.png)

Slide-over drawer pattern. Touch targets meet the 44px minimum. The top-bar hamburger icon is the single affordance to open navigation.

### Navigation open
![Navigation open](docs/PX-304.1-screenshot-nav-open.png)

Tablet view with drawer opened. Navigation is clearly separated from content; backdrop provides explicit dismiss affordance. Auto-close occurs after link selection.

### Navigation closed
![Navigation closed](docs/PX-304.1-screenshot-nav-closed.png)

Tablet view with drawer closed. Content occupies the full width. No sidebar bleed; no overlap.

## What Changed

| Area | Before | After |
|------|--------|-------|
| Frame | Floating dashboard panel | Permanent 100vh fixed frame |
| Sidebar height | Variable, did not fill viewport | 100vh, three exact zones |
| Collapse | No persistence, inconsistent widths | 280px / 72px, 200ms transition, localStorage |
| Mobile drawer | Simple overlay | Slide-over with gesture dismiss + Escape |
| Top Bar | Breadcrumbs only | Title + breadcrumbs + search + org switcher + notifications + user menu |
| Page orientation | Title buried in content | `<h1>` in sticky top bar for instant orientation |
| Content conflict | Sidebar occasionally overlapped content on resize | Flex-flow guarantee — sidebar never overlaps content |

## Verification

All states were captured and reviewed against **ReleaseFlow Design Standard v1.0** (Screen Architecture, Vertical Rhythm, Motion Language, Accessibility). No backend or domain-layer files were modified.
