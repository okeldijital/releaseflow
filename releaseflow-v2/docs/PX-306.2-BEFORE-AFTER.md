# PX-306.2 — Before/After

**Date:** 2026-06-28

---

## Operations Center

### Before
- Two H1-level elements competing: "Operations Center" and the date
- Page felt like a conventional SaaS dashboard
- No single dominant focal point
- Metrics competed with recommendations
- Activity feed was given equal visual weight as actionable sections
- Sidebar was a panel that competed with content

### After
- Date is the only H1 — page identity
- Briefing sentence provides the operational headline
- Assessment block answers "what is the current state?" with typography only
- Immediate Actions is the visual focal point — 3 max, priority-ordered
- Metrics sit directly under actions (16px gap) as evidence, not competitors
- Activity recedes — lower contrast, smaller text, no icons
- Sidebar sits transparently on the canvas

## Sidebar

### Before
- Background `bg-surface-50` — visible panel
- Border-right — separator
- Active state: `bg-primary-50/40` + primary text
- Hover: `bg-surface-100/20` + text change
- Section headers: `text-text-400 font-semibold`

### After
- Background: `transparent` — sits on canvas
- No border — no separator
- Active state: darker text only, no background
- Hover: text color only, no background
- Section headers: `text-text-300/70 font-medium` — nearly invisible
- Row radius: `rounded-md` (tighter)
- Row height: 40px (down from 44px)

## Attention Block

### Before
- Three subsections (Alerts, Blocked, Deadlines) shown in stacked cards
- Each card had its own section header with count badge
- Visual weight equal to all other sections

### After
- Single section header "Attention"
- Conditional — only when items exist
- Alerts use left-border color (critical visual signal)
- Blocked items use inline text
- Deadlines use inline text with relative dates
- Same interaction weight as before, but less chrome

---

## Activity Block

### Before
- Status dot in `text-primary-500` — drew attention
- Text in `text-text-500` with View link
- Equal visual weight to actionable sections

### After
- Status dot in `text-text-300` — recedes
- Text in `text-text-500 text-xs` with no View link
- Activity is history, not action
