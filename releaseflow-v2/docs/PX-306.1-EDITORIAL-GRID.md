# PX-306.1 — Editorial Grid

**Date:** 2026-06-28

---

## Principle

Typography—not borders, not containers—creates hierarchy. The grid is the organizational structure that supports that typography.

---

                          │  HERO (40px)      │  12 col

---

## Spacing Rhythm

| Relationship | Gap | Width | Justification |
|-------------|-----|-------|---------------|
| Page sides → content | — | 1024px max | Readable, not too wide |
| Hero → Summary | 32px | Full | Tight coupling |
| Summary → Attention | 32px | Full | Transition to detail |
| Attention → Releases | 32px | Full | Clear separation |
| Releases → Pulse | 24px | Full | Lighter separation |
| Pulse → Activity | 24px | Full | Lighter separation |
| Content → footer | 32px | Full | Page end signal |

---

## Hierarchy Levels

```
Level 1:  40px/medium/tight   → Date hero
Level 2:  14px/500            → Status line
Level 3:  13px uppercase      → Section labels (Attention, Releases, Activity)
Level 4:  14px/400            → Body content
Level 5:  12px/400            → Metadata, timestamps
```

No level should ever exceed its bounds. Level 1 appears exactly once per page.
