## Width

```
Body: 840px max (centered)
Reading column: 640px max
```

Hero briefing, assessment, actions, metrics: 640px
Active releases table: 840px (extends to reading column edge)
Activity: 640px (recedes back to reading width)

---

## Vertical Spacing

```
Hero          ── 24px ──  Assessment
Assessment    ── 16px ──  Actions
Actions       ── 20px ──  Metrics
Metrics       ── 40px ──  Releases
Releases      ── 24px ──  Activity
Activity      ──  0px ──  end
```

Tight → wider → tight pattern creates editorial flow.

## Typography Hierarchy

```
Level 1  Date              40px   medium    tight
Level 2  Briefing         15px   400       1.6
Level 3  Assessment label 10px   600       widest
Level 4  Assessment value 24px   500       tight
Level 5  Action text      15px   400       1.6
Level 6  Metric value     14px   600       tabular
Level 7  Metric label     11px   400
Level 8  Section header   10px   600       widest (uppercase)
Level 9  Timestamp       10px   600       widest (uppercase)
Level 10 Body small        12px  400
```

## Color Hierarchy

- `text-text-900` — primary content (date, values)
- `text-text-700` — secondary content
- `text-text-500` — labels
- `text-text-400` — section headers, timestamps
- `text-text-300` — ultra-quiet (sidebar headers)

## Spacing

- `mb-4` (16px) — between two related items
- `mb-5` (20px) — moderate separation
- `mb-6` (24px) — standard section break
- `mb-8` (32px) — major section
- `mb-10` (40px) — wide separation, evidence
- `mb-12` (48px) — page-level
